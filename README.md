# Melon Racer Reborn

Аркадная multiplayer-гонка для s&box, где до 12 игроков управляют физическими арбузами, проходят трассу по порядку контрольных сегментов и соревнуются за лучшее время круга. После финиша игроки голосуют за следующую карту, и новый заезд запускается без смены основной сцены игры.

## Основные системы

- **Управление арбузом.** `Melon` прикладывает физические силы для движения, прыжка и столкновений, управляет камерой от третьего лица и локальным world-space HUD игрока. При сильном ударе клиент запрашивает у host возврат на точку появления.
- **Сетевая гонка.** `GameManager` на host хранит состояние заезда, таймер, число кругов и прогресс игроков; синхронизируемые свойства передают клиентам данные для HUD. Прохождение сегментов проверяется строго по порядку, а завершение последнего сегмента закрывает круг.
- **Карты и трассы.** `MapInstance` асинхронно загружает базовую карту по package ident на каждом участнике сессии. После загрузки локальный prefab трассы добавляет одинаковые для всех визуальные меши, физику, освещение, точки появления и триггеры сегментов; игровую обработку триггеров выполняет только host.
- **Конфигурация карты.** `MapInfo` задаёт заголовок, количество кругов и список `SpawnMelon`, а имя prefab выводится из package ident активной карты. Сейчас в ротации используются `titanovsky.melon_lestoria` и `titanovsky.melon_snowball`.
- **Смена карты и голосование.** После финиша открывается десятисекундное голосование, host подсчитывает по одному голосу от соединения и выбирает победителя. Выбранное имя карты синхронизируется клиентам, после чего каждый локальный `MapInstance` выгружает старую карту и загружает новую.
- **Интерфейс.** Экранный HUD показывает время заезда, активный сегмент, круги и личные результаты. Отдельная панель голосования отображает оставшееся время, варианты карт и текущие голоса.

## Структура проекта

- `code/GameManager.cs` — host-authoritative заезд, lifecycle карт, respawn и голосование.
- `code/MapInfo.cs`, `code/SpawnMelon.cs`, `code/TriggerSegment.cs` — конфигурация трассы, точки появления и контрольные сегменты.
- `code/Melon.cs` — физика игрока, ввод, камера, столкновения и сетевой прогресс.
- `code/UI/` — HUD игрока и панель выбора следующей карты.
- `Assets/prefabs/melon_*.prefab` — игрок и локально создаваемые prefab трасс.
- `Assets/scenes/minimal.scene` — стартовая multiplayer-сцена с `NetworkHelper`, UI, камерой и `MapInstance`.

## Запуск

Стартовая сцена проекта — `scenes/minimal.scene`. Проект рассчитан на клавиатуру, 1–12 игроков и host-authoritative multiplayer.
